# Usage
```bash
$ go run main.go -h

Usage of belajar-go-kafka:
  -interactive
        enable interactive mode
  -message string
        what message you want to send
  -run string
        what you want to run (default "consumer")
  -topic string
        what topic you want to use (default "main")
```
